Prompt Engineering IDE

Vývojové prostředí prompt engineeringu.

📄️ Úvod

Prompt engineering je iterativní designová úloha. Formalizovanější oblasti iterativního designu (např. návrh loga) mají dobře zavedené nástroje, které umožňují návrhářům být efektivnější. Podobné nástroje existují i v oblasti prompt inženýrství - v poslední době byla vytvořena řada IDE (integrovaných vývojových prostředí) pro prompt inženýrství. Existuje překvapivě široká škála takových nástrojů, od výzkumných rozhraní až po profesionálně navržená IDE. V této kapitole najdete přehled některých nástrojů, které by vás jako prompt inženýra mohly zajímat.

📄️ GPT-3 Playground

GPT-3 Playground je webové rozhraní pro API GPT-3 od OpenAI. Je to pravděpodobně

📄️ Dust

Dust je nástroj pro vytváření promptů, který slouží k řetězení promptů. Poskytují webové rozhraní pro psaní promptů a jejich řetězení.

📄️ Soaked

Soaked je vývojové prostředí (IDE) pro prompt engineering, které se zaměřuje na vizuální řetězení promptů s grafickým rozhraním. V současné době je ve beta verzi, ale aktivní vývoj byl ukončen.

📄️ Dyno

Dyno je IDE pro prompt engineering, které se zaměřuje na iteraci one-shot promptu. Poskytují profesionální webové rozhraní s možností rozdělit prompt do více zobrazení pro další iteraci.

📄️ Everyprompt

Everyprompt je robustně vybavené IDE pro prompt engineering, které poskytuje profesionální rozhraní pro iteraci one-shot promptů.

📄️ Prompt IDE

Prompt IDE(@strobelt2022promptide) je výzkumný nástroj pro prompt inženýrství.

📄️ PromptSource

PromptSource(@bach2022promptsource) je výzkumný nástroj pro prompt engineering. Byl použit při vývoji BLOOM(@scao2022bloom), známého vícejazyčného LLM.

📄️ PromptChainer

PromptChainer(@wu2022promptchainer) je výzkumný nástroj pro prompt engineering. Zaměřuje se na řetězení promptů dohromady a je nejpropracovanějším výzkumným IDE, které jsem našel.

📄️ Prompts.ai

Prompts.ai poskytuje playground podobné GPT-3 Playground, ale s některými přidanými funkcemi. Již se nevyvíjí.

📄️ Snorkel 🚧

Snorkel poskytuje nástroje pro vytváření promptů a fine tuning modelů.

📄️ Multi 🚧

📄️ Human Loop

Zdá se, že Human loop nabízí playground podobné GPT-3 Playgroundu, ale s některými přidanými funkcemi. V současné době spolupracují s více uživateli z oboru.

📄️ Spellbook 🚧

Spellbook je profesionální IDE pro profesionální prompting, které vytvořila společnost Scale AI.

📄️ Kolla Prompt 🚧

Kolla Prompt je nástroj pro grafické řetězení promptů.

📄️ Lang Chain

LangChain je knihovna pro Python, která poskytuje různé funkce pro vytváření a řetězení promptů.

📄️ OpenPrompt

OpenPrompt je knihovna pro Python, která poskytuje různé funkce pro vytváření a testování promptů.

📄️ OpenAI DALLE IDE

OpenAI DALLE IDE je webová stránka pro generování obrázků pomocí DALLE-2, modelu pro převod textu na obrázek.

📄️ Dream Studio

Dream Studio je webová stránka pro generování obrázků pomocí Stable Diffusion, modelu převodu textu na obrázek.

📄️ Patience

Patience je webová stránka pro generování obrázků s několika různými modely pro převod textu na obrázek.

📄️ Promptmetheus

Promptmetheus vám pomůže sestavit, otestovat a vyhodnotit one-shot prompty pro platformu OpenAI, které transformují vstup na výstup.

📄️ PromptSandbox.io

PromptSandbox.io je vizuální programovací platforma založená na uzlech, která umožňuje navrhovat a provádět pracovní postupy pomocí rozhraní API OpenAI.

📄️ The Forge AI

The Forge AI je výkonné prompt IDE pro vytváření a monetizaci aplikací. Můžete používat různé typy konfigurovatelných vstupů, definovat podmíněné flows, generovat text a obrázky pomocí různých modelů (mnoho dalších funkcí se chystá).

📄️ Závěr

Tato stránka bude aktualizována, jakmile získám přístup k dalším IDE.
